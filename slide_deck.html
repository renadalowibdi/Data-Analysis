Ford GoBike System Data Investigation¶

by Renad Alowibdi¶

Investigation Overview¶

This presentation will answer the following questions:

  • What are the top 5 frequent places for all users types?
  • Which user type represents the bike riders in the most frequent places?
  • Which user type takes a long time for a ride in the top 5 start stations and top 5 end stations?

Dataset Overview¶

Ford GoBike is an electric bike let you go farther faster, and up hills. It's more popular in San Francisco since biking is the fastest growing mode of transportation there. Ford GoBike is expanding to the East Bay in Oakland, Emeryville and Berkeley and to San Jose in the South Bay.

Visualization 2¶

  • Which user type represents the bike riders in the most frequent places?

For the top five start stations and top-five end stations, subscribers represent most of the users than customers. This finding explains that subscribers follow routine rides for personal reasons. While perhaps, customers might be tourists and do not follow a specific destination for their rides.

Visualization 3¶

  • Which user type takes a long time for a ride in the top 5 start stations and top 5 end stations?

Although subscribers found to be the most frequent user type in the top five places for start stations and end stations, it is found that long ride durations are represented by customers. This explains that customers might be tourists who enjoy San Fransisco weather and views with a long time ride.
